19. School Search

This short notebook demonstrates how to use the schools.search() function to find schools based on partial school names or numbers.

In order to use the updated schools you must install the python fuzzy search package called thefuzz. Install with ~~~~~~ pip install thefuzz[speedup] ~~~~~~ or ~~~~~~ conda install -c conda-forge thefuzz ~~~~~~

[1]:

%load_ext autoreload
%autoreload 2
import pandas as pd
from school_data import schools, ui
